
# Olist E-Commerce Dashboard

Dashboard ini dibuat untuk menganalisis data E-Commerce publik dari Olist (Brazil). Dashboard ini menyajikan visualisasi performa penjualan produk, demografi pelanggan, dan analisis RFM (Recency, Frequency, Monetary).

## Cara Menjalankan Dashboard (Local)

1. **Clone Repository ini** (Jika mendownload dari GitHub) atau ekstrak file zip.
2. **Buka Terminal** di folder proyek ini.
3. **Install Library** yang dibutuhkan:

    ```bash
    pip install -r requirements.txt
    ```
4. **Jalankan Streamlit**:
    ```bash
    streamlit run dashboard/dashboard.py
    ```
